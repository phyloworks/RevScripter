Introduction


RevScripter is a web-based graphical tool for creating scripts in the Rev language, which then can be executed in the program RevBayes. RevBayes is a statistical inference tool for Bayesian phylogenetics.

Currently, the options available in RevScripter are limited and the tool is still very much in development. The tool can create scripts for running an unrooted analysis using nucleotide data under standard substitution models.

Source


The source code for RevScripter is hosted on GitHub: https://github.com/revbayes/revscripter
